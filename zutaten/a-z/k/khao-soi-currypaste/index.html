Thailändisch Currypaste

Wie für viele andere thailändische Currysorten gibt es auch für das nordthailändische Nudelcurry “Khao Soi” ( ข้าวซอย ) fertige Currypasten. Diese werden “Krueang Gaeng Khao Soi” ( เครื่องแกงข้าวซอย ), “Nam Prik Khao Soi” ( น้ำพริกข้าวซอย ) oder “Prik Khao Soi” ( พริกข้าวซอย ) genannt. Sehr empfehlenswert finde ich die Khao Soi Paste von Yak Thai.

Eine Khao Soi Currypaste kannst du mit etwas Aufwand auch selbst herstellen. In meinem Rezept Currypaste für Khao Soi erfährst du, wie es geht.
