Gaeng Som Currypaste

Sour Currypaste aus Zentralthailand

Currypaste Thailändisch

Gaeng Som Currypaste ist eine thailändische Fertigpaste für saurer-scharfes Curry. Diese wird meist aus getrockneten Chilis, weißem Pfeffer, Schalotten, Knoblauch, Fingerwurz und Garnelenpaste hergestellt.

Die Paste wird auf Thai “Krueang Gaeng Som” ( เครื่องแกงส้ม ), “Prik Gaeng Som" ( พริกแกงส้ม ) oder “Nam Prik Gaeng Som” ( น้ำพริกแกงส้ม ) genannt.

Gaeng Som Currypasten sind in hiesigen asiatischen Supermärkten oft unter der Bezeichnung “Sour Currypaste” zu finden.
