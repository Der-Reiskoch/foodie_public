Recipe Formatting Guidelines

This document contains formatting rules for recipe files in this directory.

HTML Recipe Format (Shopify Template Style)

When creating HTML recipe files (e.g., for Yak Thai Shopify integration), follow these formatting rules:

File Location

  • All HTML recipe variants should be stored in the _yak_thai_shopify/ subfolder
  • Path: /content/de/rezepte/_yak_thai_shopify/

File Format

  • Use .html extension for recipe files
  • Include frontmatter with YAML metadata

Ingredient List Format

  • Use table format with reversed order (ingredient first, measurement second)
  • Format:
    <table>
  <tbody>
    <tr>
      <td>Ingredient Name</td>
      <td>Measurement</td>
    </tr>
  </tbody>
</table>
  • Important: Ingredient name comes FIRST, measurement comes SECOND
  • Example:
    <tr>
  <td>Kürbis (z.B. Hokkaido)</td>
  <td>400 g</td>
</tr>

Instructions Format

  • Use numbered list (ordered list <ol>) instead of paragraph text
  • Each step as a separate <li> item
  • Format:
    <ol>
  <li>First step description.</li>
  <li>Second step description.</li>
</ol>

Structure Sections

  1. Einleitung (Introduction) - Brief description in <p> tags
  2. Zutaten (Ingredients) - Table format as described above
  3. Zubereitung (Instructions) - Numbered list format
  4. Autoren Snippet (Author Section) - Include author bio with image

Author Snippet

Always include the author snippet at the end with:

  • Author image (circular)
  • Author name and bio

Images

  • For Shopify templates: Omit recipe step images
  • Only include author profile image in the author snippet

Links

  • Link to Yak Thai products where appropriate
  • Use target="_blank" and rel="noopener" for external links
  • Format: <a href="URL" target="_blank" rel="noopener" title="Title">Link Text</a>
